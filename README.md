
Airbnb Profile Ranking System

	Compares the review of a user to all the reviews of melbourne-based
	users of Airbnb and returns where the user ranks.

    -main.py: The python file that runs the program

    -theScraper.py: It contains the functions that helps to get the profile link of Melbourne-based user.
	    This is where the scraping to get details takes place

    -getRank.py: Helps to get the rank of a user

    -review.py: Helps to get the number of Reviews from User's link.

    -photo.PNG: Picture of how the program runs.
    
    -video.ZIP: Contains short video of the program


To run code
- clone the repository
- change the pwd to this directory
- run "python main.py" in the command prompt


PS: It takes a while to get the answer.
